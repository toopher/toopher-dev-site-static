Account Recovery

It is extremely important that you have in place a way for the user to recover control of their account in case of a lost phone or other circumstances (e.g.: recover via security questions, out-of-band verification, etc.). Stay tuned, however, as we have big plans for unifying the entire two-factor authentication experience - including account recovery and complete multi-account lockdown.
