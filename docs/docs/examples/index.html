Examples

Below are some example implementations. We welcome contributions! If you email us a link to the source for a working implementation we'll add you to the list.

  • Learn to use the toopher-ruby library by extending the Rails Tutorial sample application to use Toopher.

  • For a simpler (but less flexible) integration see the Toopher Iframe Example in Flask using toopher-python.

  • The toopher-java web services library can be consumed as a JAR; see it in action with a simple project that demonstrates Toopher functionality with the toopher-java jar.
